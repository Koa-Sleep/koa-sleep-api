# Koa Sleep API

The backend REST API for [Koa Sleep](https://koasleep.dev), an open-source web dashboard for viewing and analyzing Fitbit sleep data.

Built with Node.js, Express, TypeScript, and PostgreSQL.

**Related Repository:** [Koa Sleep Client (Frontend)](https://github.com/Koa-Sleep/koa-sleep-client)

## Development Setup

### 1. Prerequisites

Ensure your system meets the following requirements:

* **Node.js:** `v20.0.0` or later.
* **npm:** `v8.0.0` or later.
* **Docker:** Required to run the local `postgreSQL` database.
